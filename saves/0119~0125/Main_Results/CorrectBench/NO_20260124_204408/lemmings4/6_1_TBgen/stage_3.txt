```python
            #   `prev_walk_left`, `prev_walk_right`: the FSM's *current stored walking direction* (1-hot, or both 0 if splattered or falling — but splattered handled above).
            #   During falling, they remain unchanged (as internal state), even though output is 0.
            #
            # So on ground rise without splatter: outputs become (stored_walk_left, stored_walk_right, 0, 0)
            #
            return (prev_walk_left, prev_walk_right, 0, 0, 0, 0)

    # --- Default fallback (should not occur per spec)
    return (1, 0, 0, 0, 0, 0)
```
